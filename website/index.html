Semi-pool.

Mining pool for BitZeny ver 2.


当プールはBitzenyウォレットの2.xへのアップグレードを強く推奨しております。

BitZenyでは、ブロック高1206400を超えますと、BitZeny 1.x が生成したブロックは2.xのウォレットではrejectされます。
弊プールに限らず、各プールや取引所では、2.xに変更済みのところがほとんどとなります。
1.xを使っている方、またはどのバージョンを使っているかわからない方は、以下の手順に沿ってウォレットをアップグレードしましょう！
ウォレット差し替え方法についてはこちらで確認可能です。

About

当プールはユーザー登録不要で掘れるBitZenyマイニングプールです。
0.05ZNY以上の採掘高があれば、60分ごとにあなたのBitZenyアドレスに払い出されます。
現在のマイニング手数料は1%です。

This pool is a BitZeny mining pool that can be digged without user registration.
If there is a mining height of 0.05 ZNY or more, it will be paid out to your ZNY address every 60 minutes.
Current mining fee is 1%.

(秘)ひねればZNYが出てくる「Daddyの蛇口」はこちら。

▼CPUの速さ ▼Diff ▼マイニング接続先
スマホなど 0.001~0.1(start 0.01) minerd -a yescrypt -o stratum+tcp://zny.daddy-pool.work:15020 -u あなたのZenyアドレス

固定Diff 0.02 minerd -a yescrypt -o stratum+tcp://zny.daddy-pool.work:15023 -u あなたのZenyアドレス
固定Diff 1 minerd -a yescrypt -o stratum+tcp://zny.daddy-pool.work:15024 -u あなたのZenyアドレス
